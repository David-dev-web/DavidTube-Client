Willkommen bei DavidTube!

Deine neue Desktop-App für YouTube. Lass uns mit der Einrichtung beginnen.

YouTube API-Schlüssel benötigt

  1. Gehe zu Google Cloud Console
  2. Erstelle ein neues Projekt oder wähle ein bestehendes
  3. Aktiviere die YouTube Data API v3
  4. Erstelle einen API-Schlüssel unter "Anmeldedaten"
  5. Füge den Schlüssel unten ein
Der API-Schlüssel wird sicher auf deinem Computer gespeichert.
